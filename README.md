# JEE – Just Enough Emotions

**JEE (Just Enough Emotions)** is an AI-powered emotional intelligence and communication coaching application designed to help users understand their attachment style, regulate emotions, and build secure relationship behaviors through guided practice and simulated conversations.

JEE combines psychology-informed frameworks with interactive AI roleplay to help users move toward secure attachment and stronger communication skills, both personally and professionally.

---

## 🌱 What JEE Does

JEE helps users:

* 🧠 Discover their attachment style (secure, anxious, avoidant, disorganized)
* 💬 Practice emotionally secure communication through AI-powered simulations
* 🎯 Improve conflict resolution skills
* ❤️ Regulate emotional reactions in high-stress conversations
* 🧩 Build self-awareness and relationship confidence

Unlike general-purpose AI tools, JEE is specifically designed around attachment theory and emotional regulation principles, creating a structured and guided improvement experience.

---

## 💡 Key Features

### 🔍 Attachment Style Assessment

A guided assessment that analyzes user responses to identify emotional and relational patterns.

### 🤖 AI Roleplay Simulator

Practice conversations such as:

* Talking to an avoidant partner
* Responding securely when feeling anxious
* Setting healthy boundaries
* Workplace conflict resolution

The AI responds dynamically, allowing users to rehearse secure communication in a safe environment.

### 📈 Emotional Growth Tracking

Users can:

* Monitor communication improvements
* Track emotional triggers
* Measure progress toward secure attachment behaviors

### 🏢 Professional Communication Mode

JEE extends beyond romantic relationships and supports:

* Leadership communication
* Stakeholder conversations
* Conflict management in teams
* Emotional intelligence development in the workplace

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone <your-repository-url>
cd jee-just-enough-emotions
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

The app will be available at:

```
http://localhost:5173
```

---

## 📦 Build for Production

To create an optimized production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
